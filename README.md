# NavJaap 🕉️

A simple, distraction-free web application for reading mantras and keeping track of your daily jaap counts. 

## The Story Behind This Project
I built this as a simple, normal project for my mother. Like many people, she sometimes had trouble remembering the exact number of times she had completed a jaap while chanting. I wanted to give her a clean, easy-to-use tool so she could focus entirely on her devotion without worrying about losing count. 

## Features
* **Jaap Counter:** A dedicated, easy-to-tap counter to track your progress.
* **Dual Language Support:** Read mantras in both Hindi and Hinglish.
* **Traditional Formatting:** Mantras are formatted beautifully in traditional two-line pairs.
* **Smart Highlighting:** The active line highlights automatically as you naturally scroll down the screen.
* **Light/Dark Mode:** For comfortable reading at any time of day.

## Contributing
Want to add a new Jaap or Mantra to the app? Contributions are very welcome! 

If there is a specific mantra you'd like to see included, feel free to fork the repository, add the Hindi and Hinglish text to the data structure, and **raise a Pull Request (PR)**. Let's build a helpful library together.
